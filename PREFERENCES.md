# PREFERENCES.md — How Jordan Likes Things

## Communication
- **Short messages** — Twitter-length when possible
- **Plain text in chat** — Not file links to open
- **Copy-paste ready** — Content Jordan can grab and send immediately
- **One action per message** — Don't overwhelm
- **Bottom line up front** — Lead with the point
- **No fluff** — "Great question" and filler = waste of time

## Actions
- **Draft first** — Jordan reviews before anything goes out
- **Confirm before sending** — External actions need approval
- **Never auto-send** — Always wait for Jordan's okay

## Format
- **Use emoji** — Helps scanning
- **Bold key info** — WhatsApp/Telegram-friendly
- **Lists over paragraphs** — Easier to read on phone
- **No markdown tables** — Use bullet lists instead

## Timing
- **Don't send walls of text at night** — Keep it brief
- **Flag urgent things** — Jordan will see it when he sees it
- **Proactive reminders** — Good, but don't spam

## Files
- **Update MISSION-CONTROL.md** — When Jordan asks what's updated
- **Keep MEMORY.md fresh** — So I remember without re-explanations
- **Commit often** — Don't lose significant changes

## What Jordan Respects
- Speed
- Directness
- Competence
- Solving problems without being asked
- Proactive suggestions

## What Frustrates Jordan
- Repeating himself (7 memory resets = extremely frustrated)
- Walls of text
- File links when he wants copy-paste
- Being kept in the dark on system status
