## HERMES-AGENT DEEP DIVE (NousResearch)

### Core Differentiator vs OpenClaw
Hermes is the "self-improving" agent — it actively creates and refines its own skills during use. OpenClaw does not yet have a native self-improvement loop.

### Key Capabilities NOT in OpenClaw
1. **Skill self-improvement during use** — agent rewrites/improves skills based on experience
2. **Autonomous skill creation** — after complex tasks, agent creates new skills procedurally
3. **FTS5 cross-session search** — full-text search of past conversations with LLM summarization
4. **Honcho dialectic user modeling** — builds deep model of user preferences across sessions
5. **Voice Mode** — real-time voice in CLI, Telegram, Discord, Discord VC
6. **15+ messaging platforms** — including Signal, Matrix, Mattermost, DingTalk, Feishu, WeCom, SMS, Email
7. **Serverless persistence** — Daytona/Modal backends hibernate when idle, cost near-zero
8. **Atropos RL training** — research-ready trajectory generation
9. **Nous Portal** — native Nous Research model provider integration

### OpenClaw Migration
Hermes can import: SOUL.md, memories, skills, API keys, TTS assets, workspace instructions, messaging settings. Command: `hermes claw migrate`
