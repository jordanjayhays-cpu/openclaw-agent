## 🔑 THUMBNAIL RULES FOR JORDAN'S SHOOT

1. **Face must dominate** — 35-40% of thumbnail real estate. The algorithm reads facial expressions as #1 signal.
2. **High contrast text** — White on teal/dark background. Min 24pt equivalent. No gradients on text.
3. **Keep it simple** — One main message. "Bali is over" OR "$655/mo" — not both competing for attention.
4. **Slight intrigue** — Don't give everything away. Leave a question in the viewer's mind.
5. **Bright, warm tones** — Bantayan is tropical paradise. Lean teal/ocean blues, gold/yellow highlights.
6. **Jordan's expression** — Go for "I know something you don't" energy. Not too excited, not bored. Confident.

## 🕐 TITLE FORMULA CHEAT SHEET

```
[PAIN POINT/CONTROVERSY]. [SOLUTION/CONTRADICTION].
[I found / The truth about / What happened when]
+ specific number if relevant
+ location if it's surprising
```

Good title structures:
- `PAIN → SOLUTION` (Bali is too expensive → Here's the alternative)
- `NUMBER DROP` ($655/month, full breakdown)
- `VS COMPARISON` (Bali vs Bantayan — the real numbers)
- `THROWBACK` (What Bali used to be like)
- `FUTURE CLAIM` (The next digital nomad hub is...)
