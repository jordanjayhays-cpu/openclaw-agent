## Recommended Location
**Bantayan Island** (Cebu, Philippines)
- $500–800/month all-in
- ~25 Mbps WiFi (workable for Zoom/calls)
- Empty beaches, peaceful, no nomad crowds yet
- Best fit: focused remote workers, not party scene

Alternatives:
- Siargao: More established nomad scene, $700–1,100/mo, ~40 Mbps
- Dumaguete: Budget pick, $400–700/mo, university town

A rough cut from his phone + basic B-roll is enough to start. He films the discovery, runs a waitlist, and uses that to negotiate.
