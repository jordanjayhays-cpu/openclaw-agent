# CommitmentClock

**A daily accountability tracker for Jordan. Log commitments, track completions, build streaks.**

## What It Does

- Log what you commit to each day (work, health, projects, life)
- Check off completed commitments
- See your win rate (completed / total commitments)
- Build daily and weekly streaks
- Review past days' performance
- Auto-categorizes commitments by keyword (work, health, project, life)

## Stack

- Single HTML file (~14KB)
- Vanilla JS + CSS (no frameworks)
- LocalStorage for persistence
- No server needed — open in any browser

## Quick Start

1. Open `index.html` in browser
2. Type a commitment → press Enter or click Add
3. Check off what you complete
4. Use ← → arrows or buttons to navigate days

## Features

- **Daily streak** — consecutive days with 100% completion
- **Weekly streak** — consecutive weeks with ≥80% win rate
- **Win rate tracker** — % of commitments completed each day
- **Category auto-tagging** — work, health, project, life (by keyword)
- **Stats dashboard** — done count, win rate %, missed count
- **Navigate history** — go back/forward by day
- **Delete commitments** — hover and click ✕

## Deploy

- **Local:** Just open `index.html`
- **Online:** Works on any static host (Netlify, Vercel, GitHub Pages, etc.)
- **Mobile:** Fully responsive, works offline
