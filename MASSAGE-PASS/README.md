# Massage Pass - What We're Building

## Customer Flow (Full)
1. Customer visits landing page
2. Selects plan (Basic/Premium/Unlimited)
3. Fills out booking form (name, email, preferred shop, time)
4. Form submits to Notion
5. You get notified (Telegram alert)
6. You coordinate with shop
7. Customer gets confirmation
